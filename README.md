##CSRF POC Generator
Simple tool  to generate CSRF Proof of Concept Code


##Requirements
Java Development Kit JDK  [Tested on jdk1.8.0] 

##How To 
### Widows
    Execute csrf-form-generator.jar

### Linux
    java -jar csrf-form-generator.jar
